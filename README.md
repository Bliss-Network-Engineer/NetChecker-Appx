This software is aimed to create a solution that keeps track of the uptime and activeness of network devices in enterprise networks, 
by countinously autonomously scanning te devices to determine whether thay are active or not, if any device goes offline, and alert is
initiated to draw the attention of the NOC operator to it, so that it can be resolved.

It helps resolve the problem of having to wait until a downtime goes out of hand before its noticed. At the very moment te downtime occurs, 
the NOC is notified immediately.

I started this project in a bid to further improve my programming skills.
Note: The project is not completed yet.
